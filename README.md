# Pantip Crawler

## Requirement
```
- Python 2.7
- GeckoDriver
- Python libs
    - selenium
    - bs4
    - requests
```

## Installation
```
git clone https://github.com/melpst/pantip_crawler.git
```

## How to crawl
```
python pantip_crawler.py <tag>
```

## How to test

```
python -m doctest test_classifier.txt
```
